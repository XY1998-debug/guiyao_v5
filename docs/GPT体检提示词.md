# 归爻（Gui Yao）量化系统 — GPT 全面体检提示词

你是最优秀的量化系统架构师。请以审计专家的身份，对以下量化系统进行一次全面的技术体检。不要夸奖，不要避讳，指出痛处。

## 系统概况

归爻是一个面向 A 股散户的全自动量化辅助交易系统。10 万本金（银河5万ETF轮动 + 国泰海通5万个股突破）。纯做多，不融资融券。

### 硬件约束（不可逾越）
- 本地：Windows 10, 12GB RAM, NUMBA_NUM_THREADS=2
- 云端：Ubuntu 24.04, 3.6GB RAM, NUMBA_NUM_THREADS=2
- 数据库：仅 SQLite + Parquet。禁止 PostgreSQL/MySQL/Redis/Kafka
- 依赖：仅 numpy, numba, polars, mootdx, requests, pyyaml

### 当前架构

三层脑:
  宏观脑 — 4维评分(趋势40+波动30+宽度30)，bear时一票否决
  中观脑 — 22只ETF周频RS排名，Top3持仓，Top5缓冲带，全负RS→国债安全垫
  微观脑 — 个股突破(3x放量+阳线+日内高位>=70%)，10%试错仓

V3 Numba 引擎 — @njit(parallel=True)，T+1标记，跌停判定，佣金模型(万三+千一+最低5元)
  性能：200宇宙x210股 < 0.05秒/日，384次WF < 1分钟

### WF验证结果
ETF轮动(22只中观脑): 夏普+0.37 年化7.1% 回撤14.3%
个股突破: 盈亏比4.8 夏普+0.53 回撤18.6% 胜率24%

### 最新更新 V5.P1 (2026-07-17)
执行层已补齐：Gate(宏观否决) + Calculator(动态止损止盈 ATRx2xgamma + 跟踪止盈浮盈三档 + 宏观状态系数) + Injector
5套系统统一配置(训练集/虚拟宇宙/模拟盘/影子账本/实盘)
价格告警已集成到盯盘(5秒轮询触发推企微)
T+1磨损惩罚、涨跌停市场区分(主板/创业板/科创板)、新老划断
训练数据1,183条闭环交易(v4.0基线)

### 当前数据
训练数据: market_v5.parquet(210只x827天), etf_v5.parquet(22只x800天), trades_synthetic.parquet(1183条)
虚拟宇宙: 960组参数每日真行情竞跑
云端: 企业微信回调服务(端口80), 盘中盯盘(5秒轮询), 里程碑提醒(crontab 4节点)

2. 极端行情样本未覆盖(2015股灾/2024微盘踩踏)
3. NSGA-II多目标优化延后(V5.5+, 2026-09评估)
4. 同花顺自动添加自选接口参数未确认
5. 腾讯行情API可用未接入
