## 一、系统总览

归爻是一个面向 A 股散户的全自动量化辅助交易系统。核心架构为**三层脑 + 双策略并行**，总代码量约 2,100 行，部署于 Windows 10 本地 + Ubuntu 24.04 云端。

### 1.1 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│  宏观脑（市场状态）→ 一票否决权                               │
│    ├─ bear → 全仓国债，中观/微观全部静默                       │
│    ├─ chop → ETF执行，个股降级观察                             │
│    └─ bull → 全信号执行                                       │
│         ↓                                                    │
│  中观脑（ETF轮动）→ 22只ETF，周频RS排名，Top3持仓，Top5缓冲    │
│    ├─ RS = 20日涨幅×0.7 + 60日涨幅×0.3                       │
│    ├─ 缓冲带：前5不换（减40%无效换仓）                         │
│    └─ 安全垫：全负RS → 国债ETF                                │
│         ↓                                                    │
│  微观脑（个股突破）→ 日频3x放量，10%试错仓                       │
│    ├─ 选股：日均成交>5亿，20日振幅Top30                        │
│    ├─ 止损：-5%硬止损                                         │
│    └─ 加仓：浮盈+5% → 40%                                     │
│                                                             │
│  V3 Numba 引擎（并行回测核心）                                 │
│    ├─ 200宇宙×210股 < 0.05秒/日                              │
│    ├─ 384次回测 < 1分钟                                       │
│    └─ 1,183条虚拟交易已生成                                    │
└─────────────────────────────────────────────────────────────┘
```
