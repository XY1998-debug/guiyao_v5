# 归爻 V5 — 执行层（价格建议 + 统一规则）专家咨询

> 附件：参评报告书 GY-TR-2026-01（系统全貌）
> 背景：归爻 V5 的三层脑架构已封版。当前缺口为执行层——系统只输出方向与仓位，未输出具体价格建议。

---

## 一、现状

### 1.1 已有能力

| 层 | 输出 | 状态 |
|:--:|------|:----:|
| 宏观脑 | 市场状态（bear/chop/bull） | ✅ |
| 中观脑 | ETF 轮动信号（Top3 + 缓冲带） | ✅ |
| 微观脑 | 个股突破信号（3x放量，止损触发） | ✅ |
| 虚拟宇宙 | 960组参数在真实行情中竞跑 | ✅ |
| 影子账本 | AI vs 实盘对比 | ✅ |
| 训练数据 | trades_synthetic.parquet | ✅ |

### 1.2 缺失的执行层

当前信号仅有：买入/卖出方向 + 仓位比例（10%试错/40%加仓/-5%止损）

缺失：建议买入价、止损价设定规则、止盈规则、滑点估算、非对称仓位执行逻辑。

### 1.3 多组件一致性问题

训练集、虚拟宇宙、影子账本、模拟盘、实盘五套系统的执行规则互不相同，绩效不可横向对比。

---

## 二、核心问题（不预设答案）

### 2.1 止损/止盈/挂单价的最优设计

- 参数是否应随市场状态/持仓阶段自适应？
- 自变量是什么？（ATR、波动率、三脑输出、持仓天数、浮盈比例……）
- 非对称仓位如何在执行层体现？

### 2.2 多组件统一方案

- 五套系统应共享一套执行规则，还是分层？
- 共享的最小公共接口是什么？
- 不共享时的偏差如何处理？

### 2.3 模拟盘最优设计

- 960虚拟宇宙提供全覆盖后，模拟盘还有价值吗？
- 保留几个？各承担什么角色？是否与虚拟宇宙联动？
- 注意：数量不是预设的。如果最优是3个、8个、或0个，请论证。

### 2.4 三脑输出在执行层的作用

- 宏观脑应调节执行系数吗？
- 中观脑对个股执行层有作用吗？
- 微观脑和执行层应严格分离吗？

|:----:|------|
