## 四、宏观脑状态机重写（终裁版）

`regime.py` — 完整的 `detect()` 方法（约40行）

```python
# 判定顺序（优先级从高到低）
1. 宽度一票否决：width < 12 → 强制Bear
2. Bear触发：total < 25 → Bear
3. Bear回升：prev=Bear → Chop（不直接回Bull）
4. Bull触发：total≥70 或 ma3≥70 → Bull
5. Bull退出：prev=Bull 且 total<65 → Chop
6. 其余情况：Chop
```

### 趋势分割波动修正

| 条件 | 波动评分 |
|:----|:--------:|

| 趋势<15 | 封顶20 |
