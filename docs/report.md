## 一、系统概览

归爻是一个面向 A 股散户的全自动量化辅助交易系统。核心架构为**三层脑 + 双策略并行**，总代码量约 2,100 行，部署于 Windows 10 本地 + Ubuntu 24.04 云端。

| 指标 | 值 |
|------|-----|
| 总模块数 | 14 个核心 Python 文件 |
| 策略线 | 2 条（ETF 轮动 + 个股突破）+ 虚拟宇宙960组 |
| 实盘资金 | 10 万元（银河 5万 + 国泰海通 5万） |
| 最高单笔仓位 | 15%（个股）、33%（ETF） |
| 数据频率 | 日频（盘前选股）+ 5秒（盘中盯盘） |
| WF 验证方式 | 滚动窗口，训练/测试独立划分 |

## 三、V3 引擎（回测核心）

### 3.1 Numba 平行宇宙（Sandbox）

V3 引擎是归爻的核心资产，发布于 2026年7月。

**核心创新**：将 Numba `@njit(parallel=True)` 编译引入回测引擎，实现**多宇宙并行计算**。

```python
@njit(parallel=True, cache=True)
def backtest_kernel_v3(
    prices,          # (S,) float64 — 当日各股票价格
    signals,         # (U, S) int8  — 100 组参数信号
    atrs,            # (S,) float64 — 当日 ATR
    limit_down,      # (S,) float64 — 跌停价
    total_capital,   # (U, 1) float64 — 本金（只读，用于算股数）
    cash,            # (U, 1) float64 — 现金（可扣减）
    positions,       # (U, S) int32 — 持仓股数
    cost_basis,      # (U, S) float64 — 持仓成本
    bought_today_in, # (U, S) int8 — 昨日买入标记（T+1保护）
    bought_today_out,# (U, S) int8 — 今日买入标记
    yesterday_limit_down, # (S,) bool — 昨日跌停标记
    n_univ,          # int
    n_stocks,        # int
):
    for u in prange(n_univ):
        for s in range(n_stocks):
            # 卖出逻辑：signal=-1 + 持仓 > 0 + 非昨日买入 + 非跌停
            # 买入逻辑：signal=1 + 现金够 + T+1
            # 成本扣减：含佣金万三+千一印花税+最低5元
            # 跌停判定：abs(close - limit_down) < 0.01 → 不可卖出
            # 连续跌停解锁：yesterday_limit_down=1 且今日非跌停 → 卖出价×0.98
            pass
```

### 3.2 性能

| 指标 | 值 |
|------|-----|
| 100宇宙 × 210股票 | < 0.05 秒/日 |
| 827 天全期 | < 40 秒 |
| 内存占用 | < 3 MB（矩阵） |
| Numba 编译后首次调用 | < 2 秒 |
| 200宇宙 × 210股 × 4窗口 × 2段 | **384 次回测 < 1 分钟** |

### 3.3 精度约束

| 规则 | 说明 |
|------|------|
| 分离 total_capital / cash | 股数计算用本金（不变），买卖扣减用现金（可变） |
| T+1 标记 | 买入当日标记，次日才能卖（Python层管理跨天传递） |
| 跌停判定 | `abs(prices[s] - limit_down[s]) < 0.01` 视为不可卖出 |
| 首日开板惩罚 | 连续跌停后首次可卖日 → 卖出价 × 0.98 |
| 佣金模型 | 买：0.03% + 最低5元。卖：0.03% + 0.1%印花税 + 最低5元 |
| 最小交易单位 | 100 股（1手），向下取整 |
| 资金保护 | 买不起 100 股则跳过，禁止现金变负 |

## 八、工程特性

| 特性 | 说明 |
