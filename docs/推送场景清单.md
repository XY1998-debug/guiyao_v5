# 归爻 V5 推送场景全景清单

## 一、定时推送（4 类）

| 编号 | 时间 | 名称 | 触发条件 | 紧急度 |
|:----:|:----:|------|---------|:------:|
| T1 | 09:10 | 早间简报 | 每日开盘前 | ⭐ |
| T2 | 09:10 | 市场研报 | 每日开盘前（附T1后） | ⭐ |
| T3 | 11:35 | 午间策略调整 | 每日午盘 | ⭐⭐ |
| T4 | 16:00 | 收盘复盘+持仓 | 每日收盘后(等ETF清算) | ⭐ |
| T5 | 周日20:00 | 周报 | 每周日 | ⭐ |

## 二、事件触发（14 类）

| 编号 | 名称 | 触发条件 | 紧急度 |
|:----:|------|---------|:------:|
| E1 | 个股突破信号 | 选股池某股票放量>3x | ⭐⭐ |
| E2 | 止损告警 | 持仓股亏损达-5% | 🔴 高 |
| E3 | 浮盈加仓确认 | 持仓股浮盈达+5% | ⭐⭐ |
| E4 | 极端流动性熔断 | breadth<10% + 跌停>300 | 🔴 最高 |
| E5 | 大盘单日暴跌 | 大盘跌>3% | 🔴 高 |
| E6 | 持仓股异动 | 单日涨跌幅>5% | ⭐⭐ |
| E7 | 冷静期开始 | 某股触发止损进入3天冷静期 | ⭐ |
| E8 | 冷静期结束 | 冷静期到期解除 | ⭐ |
| E9 | ETF轮动调仓日 | 周五收盘调仓信号 | ⭐⭐ |
| E10 | 数据源异常告警 | mootdx/akshare连接失败 | ⭐ |
| E11 | 系统启动失败 | 9:00数据未按时加载 | 🔴 高 |
| E12 | 连续止损N次 | 当日连续止损2次以上 | ⭐⭐ |
| E13 | 午间异动修正 | 早盘突发利空/利多 | 🔴 高 |
| E14 | 影子账本对比异常 | 实盘连续跑输AI两周 | ⭐⭐ |

## 三、用户主动查询（6 类）

| 编号 | 用户输入 | 名称 | 回复内容 |
|:----:|---------|------|---------|
| U1 | "持仓" 或 "仓位" | 实时持仓 | 当前持仓+盈亏+可用现金 |
| U2 | "600519" 或 "贵州茅台" | 个股分析 | 日线+因子+建议 |
| U3 | "信号" 或 "今天买什么" | 今日信号 | 当日候选股+理由 |
| U4 | "冷静期" | 冷静期列表 | 当前在冷静期的股票+剩余天数 |
| U5 | "参数" | 策略参数 | 当前WF最优参数 |
| U6 | "复盘" 或 "周报" | 复牌/周报 | 最近一周统计 |
